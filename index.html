    Transpiler Mate

    A small and light yet powerful API + CLI to extract Schema.org/SoftwareApplication Metadata from an annotated CWL document and publish it as a Record on Invenio RDM.

    Pre-requisites

    You must own an authentication Token to pusblish on Invenio, see how to create a new Token.

    Installation

    pip install transpiler-mate

    CLI Usage

    $ transpiler-mate --help
Usage: transpiler-mate [OPTIONS] SOURCE

Options:
  --invenio-base-url TEXT  The Invenio server base URL
  --auth-token TEXT        The Invenio Access token
  --help                   Show this message and exit.
  ```

i.e.
    $ transpiler-mate --invenio-base-url https://sandbox.zenodo.org --auth-token= ./docs/pattern-1.cwl
