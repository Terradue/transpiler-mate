# Tutorials

Tutorials are learning-oriented guides that walk through a complete workflow step by step.

The current documentation set does not yet include a maintained end-to-end tutorial with sample CWL input and expected generated output. Until that is added, start with the [metadata generator](../how-to-guides/metadata-generator.html) and then follow one of the task guides in [How-to guides](../how-to-guides/index.md).
